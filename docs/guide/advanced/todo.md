# TODO機能

チャットのメッセージをTODOリストとして管理できます。

## TODOの追加

1. メッセージを**長押し**（PCでは右クリック）
2. コンテキストメニューから「TODO」を選択
3. TODOタグを選択、または新しいタグを作成

## TODOの管理

検索ページのTODOフィルタから、TODOの一覧を管理できます。

| 操作 | 説明 |
|---|---|
| チェックボックス | 完了/未完了の切替 |
| 優先度（★） | 優先度の設定 |

完了したTODOは取り消し線と薄い色で表示されます。

!!! tip
    TODOは既存のタグシステムを利用しているため、タグでの分類・絞り込みが可能です。
