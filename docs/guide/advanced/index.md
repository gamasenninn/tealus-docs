# 応用機能

Tealus の応用的な機能を説明します。

- [メッセージ検索](search.md) — キーワードで全ルーム横断検索
- [リアクション](reaction.md) — 絵文字でメッセージにリアクション
- [メンション](mention.md) — @ユーザー名で特定メンバーに通知
- [スタンプ](stamp.md) — AIが自動生成するオリジナルスタンプ
- [TODO機能](todo.md) — メッセージをTODOリストとして管理
- [音声・ビデオ通話](call.md) — WebRTCによるグループ通話・PTT
- [TTS 読み上げ](tts.md) — AIの回答を音声で自動読み上げ
