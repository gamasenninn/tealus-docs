# 音声・ビデオ通話

WebRTC（mediasoup SFU）による高品質な音声・ビデオ通話が利用できます。

## 通話の開始

1. トーク画面の通話ボタンをタップ
2. 確認ダイアログで音声/映像の設定を確認
3. 「発信」をタップ

相手には着信通知が表示されます。

## 通話の受信

着信があると画面に着信UIが表示されます。「受話」をタップすると通話が始まります。

## グループ通話

グループルームでは複数人が同時に通話に参加できます。

- 途中参加・途中退出が可能
- 参加人数に応じてグリッドビュー/スピーカービューが自動切替

!!! note "同時接続数"
    家庭回線の場合、同時4〜6人が現実的な上限です。

## Web トランシーバー（PTT: Push-to-Talk）

トランシーバーのようにボタンを押している間だけ音声を送信するモードです。

### 有効化

1. プロフィール設定でトランシーバー機能をONにする
2. 対応ルームのチャット画面にPTTボタンが表示される

### 使い方

- PTTボタンを**長押し**している間だけ送信
- ボタンを離すと送信停止
- 受信は常時

PTTで送信した音声は、自動的に音声メッセージとしてチャットにも投稿されます。後から聞き返したり、文字起こしを確認したりできます。

!!! info "通話との違い"
    PTTは映像なし・音声のみで、通常の通話より帯域消費が少ないです。
