# TTS 読み上げ

AIエージェントの回答を音声で読み上げる機能です。Aivis Cloud API による高品質な日本語音声合成で、レイテンシ 0.3〜0.5秒のリアルタイム読み上げを実現しています。

## 仕組み

```
AIが回答テキストを生成
  → Aivis Cloud API で音声合成（WAV）
  → ffmpeg で Opus エンコード → RTP 送信
  → mediasoup PlainTransport 経由
  → ブラウザのトランシーバーで再生
```

テキストメッセージの送信と音声合成は並行して処理されるため、チャットへのテキスト表示がブロックされることはありません。

## 使い方

### 1. 読み上げを有効にする

プロフィール設定の「AI回答の読み上げ」をONにします。

### 2. メッセージを送信する

AIエージェントがいるルームでメッセージを送信すると、トランシーバーが自動接続されます。

### 3. 音声で回答を聞く

AIの回答がチャットに表示されると同時に、トランシーバーから音声が流れます。

### 4. 自動切断

30秒間新しい回答がなければ、トランシーバーが自動的に切断されます。読み上げ中は切断されず、完了を待ってから切断されます。

## テキストの前処理

読み上げ前にテキストは以下の前処理が行われます。

| 処理 | 内容 |
|---|---|
| Markdown除去 | 見出し・太字等のMarkdown記法を除去 |
| URL省略 | URLは「URL省略」に置換 |
| コード省略 | コードブロックは「コード省略」に置換 |
| 文字数制限 | 500文字を超える部分は切り詰め |

## 音声モデル

ルームごとに読み上げの音声（声の種類）を選択できます。設定は管理者がダッシュボードのルーム設定から行います。

### 選択可能なモデル

| モデル名 | 声の特徴 |
|---|---|
| 凛音エル（デフォルト） | 青年女性 |
| まお | 青年女性 |
| にせ | 青年男性 |
| まい | 青年女性 |
| 阿井田 茂 | 中年男性 |
| fumifumi | 成人男性 |
| morioki | 成人女性 |
| ろてじん | 長老ボイス（老年男性） |
| 花音 | 青年女性 |
| コハク | 青年女性 |

!!! tip "声はエージェントの性格の一部"
    音声モデルはプロンプトと同じレイヤで管理されています。エージェントのキャラクター設定に合わせて声を選ぶことで、より自然なコミュニケーションが実現できます。

## 長文メッセージの折りたたみ

300文字を超えるメッセージは自動的に折りたたまれ、「もっとみる」ボタンが表示されます。クリックで展開、「閉じる」で再折りたたみできます。これはTTSとは独立した表示機能ですが、AIの長文回答が読みやすくなります。
