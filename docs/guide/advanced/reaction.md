# リアクション

メッセージに絵文字でリアクションを付けられます。

## リアクションの付け方

1. メッセージを**長押し**（PCでは右クリック）
2. コンテキストメニュー上部の絵文字パレットから選択

使用可能な絵文字:

| 絵文字 | 意味 |
|---|---|
| 👍 | いいね |
| ❤️ | ハート |
| 😂 | 笑い |
| 🎉 | お祝い |
| 👀 | 確認 |
| 🙏 | お願い・感謝 |

## リアクションの表示

メッセージの吹き出し下部に「絵文字 + 件数」が表示されます（例: 👍 3）。

## リアクションの取り消し

同じ絵文字をもう一度タップすると、リアクションが取り消されます（トグル動作）。
