# メッセージ検索

トーク内のメッセージをキーワードで検索できます。

## 検索方法

1. ボトムナビゲーションまたはヘッダーの検索アイコンをタップ
2. 検索バーにキーワードを入力
3. 検索結果が表示される

## 検索結果の見方

検索結果には以下の情報が表示されます。

- メッセージの抜粋テキスト（キーワードがハイライト表示）
- ルーム名
- 送信者名
- 送信日時

## メッセージへのジャンプ

検索結果をタップすると、該当メッセージの位置に自動でジャンプします。

## 検索対象

- テキストメッセージ
- 音声メッセージの文字起こしテキスト
- すべての参加ルームが横断検索の対象
