# スタンプ

AIが自動生成するオリジナルスタンプを使えます。

## スタンプの送信

1. 入力欄のスタンプボタンをタップ
2. スタンプパックを選択
3. 使いたいスタンプをタップして送信

## スタンプパックの作成

AIにプロンプトを1行入力するだけで、16枚のスタンプパックが自動生成されます。

1. スタンプ画面の「作成」ボタンをタップ
2. プロンプトを入力（例:「柴犬が仕事してるスタンプ」）
3. AIがプロンプトを詳細化 → 画像生成 → 16枚に分割
4. スタンプパックが完成

生成されたスタンプパックは**全社に公開**され、誰でも使用できます。

!!! info "スタンプ仕様"
    - 1パック **16枚**（4×4グリッド）
    - LINEスタンプ規格準拠（370×320px PNG透過）
    - 生成制限: 1ユーザー1日 **3パック**まで（管理者は無制限）
