# インストールと初期設定

Tealus は PWA（Progressive Web App）として動作します。アプリストアからのインストールは不要で、ブラウザからすぐに使い始められます。

## アクセス方法

管理者から共有された URL にブラウザでアクセスしてください。

## ホーム画面に追加（推奨）

PWA をホーム画面に追加すると、アプリのように利用できます。

### スマートフォン（iOS / Android）

1. ブラウザで Tealus の URL を開く
2. 共有ボタン（iOS）またはメニュー（Android）をタップ
3. 「ホーム画面に追加」を選択

### PC（Chrome / Edge）

1. ブラウザで Tealus の URL を開く
2. アドレスバーのインストールアイコンをクリック
3. 「インストール」を選択

## 初回ログイン

管理者から発行された**ユーザーID**と**パスワード**でログインします。詳しくは[ログイン](basic/login.md)を参照してください。
