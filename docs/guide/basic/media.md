# メディア送信

画像・動画・ファイルをチャットで送信できます。すべてのファイルはサーバーに保存されるため、端末の容量を消費しません。

## 画像の送信

1. 入力欄のファイル添付ボタンをタップ
2. 画像ファイルを選択（複数選択可）
3. 送信ボタンをタップ

画像はサムネイルが自動生成され、トーク画面にはサムネイルが表示されます。

## 動画の送信

画像と同様の手順で動画ファイルを送信できます。

## ファイルの送信

PDF、Excel、Word などの一般的なファイルも送信可能です。

!!! warning "ファイルサイズ制限"
    動画を含むメディアファイルの上限は約100MBです。

## 複数画像のグリッド表示

複数枚の画像を同時に送信すると、枚数に応じたグリッドレイアウトで表示されます。

| 枚数 | レイアウト |
|---|---|
| 1枚 | 大きく1枚表示 |
| 2枚 | 横に2列 |
| 3枚 | 上に1枚大 + 下に2枚小 |
| 4枚 | 2×2グリッド |
| 5枚以上 | 3列グリッド |

## 画像ビューア

画像をタップすると、全画面の画像ビューアが起動します。背景が暗くなり、画像が原寸で中央に表示されます。タップまたは「×」ボタンで閉じます。

## 画像ダウンロード

- **1枚ダウンロード**: 画像ビューアの拡大表示中に「↓」ボタン
- **一括ダウンロード**: 画像グリッドのダウンロードアイコン

## 外部アプリからの共有（Share Target）

スマホの他のアプリから「共有」機能で Tealus にテキスト・画像・動画を送信できます。

1. 他のアプリで「共有」ボタンを押す
2. 共有先から「Tealus」を選択
3. 送信先のルームを選んで送信

!!! note
    Share Target 機能を使うには、Tealus をPWAとしてホーム画面にインストールしている必要があります。
