# グループ

## 1対1トーク

特定のユーザーと1対1でチャットできます。

1. トーク一覧画面の「新規作成」ボタンをタップ
2. ユーザーを選択
3. 自動的に1対1のダイレクトルームが作成される

!!! note
    同じ相手との1対1ルームは重複して作成されません。既にルームが存在する場合は、そのルームが開きます。

## グループの作成

複数人でチャットするグループを作成できます。

1. トーク一覧画面の「新規作成」ボタンをタップ
2. 「グループ作成」を選択
3. グループ名を入力
4. メンバーを選択
5. 「作成」をタップ

## 単独グループ（メモ機能）

自分だけのグループを作成すると、個人メモとして活用できます。メンバーを選択せずにグループを作成してください。

## グループ名・アイコンの変更

グループ管理者は、グループ名やアイコン画像を変更できます。

1. トーク画面のヘッダーをタップしてルーム設定を開く
2. グループ名を編集、またはアイコン画像をアップロード

## メンバー管理

グループ管理者は以下の操作が可能です。

| 操作 | 説明 |
|---|---|
| メンバー追加 | 新しいメンバーをグループに招待 |
| メンバー除外 | メンバーをグループから除外 |
| 管理者変更 | 他のメンバーに管理者権限を付与 |

一般メンバーは、自分自身の**退会**のみ可能です。

操作が行われると、トーク内にシステムメッセージ（「○○さんが参加しました」等）が自動投稿されます。

!!! warning
    グループの削除機能はありません。全員が退会するとトーク一覧から表示されなくなります。
