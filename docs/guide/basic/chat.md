# トーク（チャット）

Tealus のメイン機能であるチャットの使い方を説明します。

## メッセージの送信

1. トーク一覧から相手またはグループをタップしてトーク画面を開く
2. 画面下部の入力欄にメッセージを入力
3. **Ctrl+Enter**（Macは Cmd+Enter）で送信

!!! tip "改行と送信"
    **Enter** キーは改行になります。送信は **Ctrl+Enter** です。入力欄は内容に応じて最大6行分まで自動拡張します。

## メッセージの受信

メッセージはリアルタイムで配信されます。トーク画面を開いている間は、新しいメッセージが即座に表示されます。

トーク画面を閉じている場合は、トーク一覧に**未読バッジ**（未読数）が表示されます。

## 既読表示

Tealus には2種類の既読表示があります。

- **トーク一覧**: 未読メッセージの数がバッジで表示されます
- **トーク画面**: 各メッセージに既読数が表示されます（例: 既読 3）

トーク画面を開くと、そのルームのメッセージは自動的に既読になります。

## 未読区切り線

前回の閲覧以降に届いた未読メッセージがある場合、「ここから未読」の区切り線が自動で表示されます。どこまで読んだかが一目でわかります。

## リプライ（引用返信）

特定のメッセージに対して返信できます。

1. 返信したいメッセージを**長押し**（PCでは右クリック）
2. コンテキストメニューから「リプライ」を選択
3. 入力欄の上部に引用元メッセージが表示される
4. メッセージを入力して送信

送信されたリプライには、引用元メッセージの内容が吹き出し内に表示されます。

## メッセージの編集

送信済みのメッセージを修正できます。

1. 自分のメッセージを**長押し**（PCでは右クリック）
2. コンテキストメニューから「編集」を選択
3. テキストを修正して確定

編集されたメッセージには「(編集済み)」の表示が付きます。過去の編集履歴は「履歴ボタン」から確認できます。

!!! note "編集権限"
    デフォルトでは自分のメッセージのみ編集可能です。ルーム管理者がメンバー全員に編集を許可する設定も可能です。

## メッセージの削除

1. 自分のメッセージを**長押し**（PCでは右クリック）
2. コンテキストメニューから「削除」を選択
3. 確認ダイアログで「削除」をタップ

削除されたメッセージは「メッセージが削除されました」と表示されます。他のメンバーのメッセージは削除できません。

## コンテキストメニュー

メッセージを長押し（PCでは右クリック）すると、以下の操作が表示されます。

| 操作 | 対象 | 説明 |
|---|---|---|
| コピー | テキストメッセージ | テキストをクリップボードにコピー |
| リプライ | すべてのメッセージ | 引用返信 |
| 編集 | 自分のメッセージ | テキストを修正 |
| 削除 | 自分のメッセージ | メッセージを削除 |
| TODO | すべてのメッセージ | TODOタグを付与 |

## 吹き出しの全幅展開

AIアシスタントの長文回答など、幅広い内容を読む際に便利な機能です。

- 吹き出しを**ダブルタップ**すると画面幅いっぱいに展開
- もう一度ダブルタップすると元の幅に戻る

## リンクプレビュー

メッセージ内のURLは自動的にリンクプレビュー（OGP）が表示されます。Webページのタイトル・説明・画像がプレビューカードとして表示されます。

## 日付区切り

メッセージの間に日付が変わるポイントで「── 4月23日 ──」のような区切り線が自動挿入されます。

## タイピング表示

相手がメッセージを入力中のとき、「入力中...」の表示がリアルタイムで表示されます。
