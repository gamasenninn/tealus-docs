# 表示設定

## 文字サイズ

トーク画面のメッセージ文字サイズを3段階で変更できます。

| サイズ | ピクセル |
|---|---|
| 小 | 13px |
| 中（デフォルト） | 15px |
| 大 | 17px |

プロフィール画面の文字サイズ設定から変更します。変更は即座に反映されます。

!!! note
    文字サイズの設定はトーク画面のメッセージのみに適用されます。ヘッダーやメニューの文字サイズは変わりません。

## カラーテーマ

画面全体のカラーを変更できます。

| テーマ | 説明 |
|---|---|
| Teal（デフォルト） | ティールカラー |
| Ocean | 青系 |
| Sunset | 暖色系 |
| Dark | ダークモード |

設定画面またはヘッダーからテーマを切り替えます。変更は即座に画面全体に反映されます。

## Markdownプレビュー

メッセージのMarkdown記法（見出し・太字・コードブロック等）をレンダリングして表示する設定です。ユーザー設定から切り替えられます。

!!! note "設定の保存先"
    文字サイズ・カラーテーマ・Markdownプレビューの設定は端末のローカルストレージに保存されます。別の端末からログインした場合は再設定が必要です。
