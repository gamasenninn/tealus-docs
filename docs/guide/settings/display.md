# 表示設定

## 文字サイズ

トーク画面のメッセージ文字サイズを3段階で変更できます。

| サイズ | ピクセル |
|---|---|
| 小 | 13px |
| 中（デフォルト） | 15px |
| 大 | 17px |

プロフィール画面の文字サイズ設定から変更します。変更は即座に反映されます。

!!! note
    文字サイズの設定はトーク画面のメッセージのみに適用されます。ヘッダーやメニューの文字サイズは変わりません。

## Markdownプレビュー

メッセージのMarkdown記法（見出し・太字・コードブロック等）をレンダリングして表示する設定です。ユーザー設定から切り替えられます。

## 長文の折りたたみ

300文字を超えるメッセージは自動的に折りたたまれ、「もっとみる」ボタンが表示されます。クリックで展開、「閉じる」で再折りたたみできます。

## Wake Lock（画面スリープ防止）

以下の場面で画面のスリープが自動的に防止されます。

- 音声メッセージの録音中
- 音声メッセージの連続再生中
- アプリパネル表示中（ルーム設定で Wake Lock を有効にした場合）

!!! note "設定の保存先"
    文字サイズ・Markdownプレビューの設定は端末のローカルストレージに保存されます。別の端末からログインした場合は再設定が必要です。
