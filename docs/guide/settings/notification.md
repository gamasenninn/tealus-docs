# 通知設定

## プッシュ通知

Tealus はPWAのService Workerを利用したプッシュ通知に対応しています。

### 通知の有効化

初回利用時にブラウザから通知許可を求められます。「許可」を選択するとプッシュ通知が有効になります。

### 通知されるイベント

- 新着メッセージ
- メンション（`@名前`）
- 通話の着信

メンションされた場合は、通常のメッセージよりも優先度の高い通知が送信されます。

## 通知音

新しいメッセージを受信すると通知音が鳴ります。

## Wake Lock

画面のスリープを防止するWake Lock設定が利用可能です。アプリパネルの設定から有効化できます。長時間のモニタリングや音声の連続再生時に便利です。
