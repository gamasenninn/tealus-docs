# 通知設定

## プッシュ通知

Tealus はPWAのService Workerを利用したプッシュ通知に対応しています。

### 通知の有効化

初回利用時にブラウザから通知許可を求められます。「許可」を選択するとプッシュ通知が有効になります。

### 通知されるイベント

- 新着メッセージ
- メンション（`@名前`）
- 通話の着信

メンションされた場合は、通常のメッセージよりも優先度の高い通知が送信されます。

## ホーム画面アイコンの未読バッジ（PWA App Badge、v0.2.4〜）

PWA としてホーム画面に追加している場合、**ホーム画面アイコンに未読件数のバッジが表示**されます。Tealus を開かなくても、未読の有無と件数が一目で分かります。

!!! info "対応環境"
    OS / ブラウザの App Badge API 対応状況に依存します。

    - **iOS / iPadOS**: Safari + ホーム画面に追加した PWA で対応
    - **Android**: Chrome + ホーム画面に追加した PWA で対応
    - **デスクトップ（macOS / Windows）**: Chrome / Edge でインストールした PWA で対応

    非対応環境では、バッジは表示されませんが他の通知機能（プッシュ通知 / 通知音）は通常通り動作します。

### 動作の概要

1. Tealus を開いていない時に未読メッセージが届く
2. Service Worker が未読件数を OS にレポート
3. ホーム画面アイコンの右上に数字バッジが出現
4. Tealus を開いて既読化するとバッジは自動で消える

## 通知音

新しいメッセージを受信すると通知音が鳴ります。プロフィール設定からON/OFFを切り替えられます。
