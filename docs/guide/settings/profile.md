# プロフィール設定

## 表示名の変更

プロフィール画面から表示名を変更できます。変更後は即座に全ルームに反映されます。

## プロフィール画像

プロフィール画面からアバター画像をアップロードできます。アップロードされた画像はサーバーに保存され、トーク画面やメンバー一覧に表示されます。

## ステータスメッセージ

自分の状態を一言で伝えるステータスメッセージを設定できます。

## パスワード変更

現在のパスワードと新しいパスワードを入力して変更します。

## 音声再生レベル

トランシーバー音声の再生音量をスライダー（0〜100%）で調整できます。デフォルトは80%です。

## AI回答の読み上げ

AIエージェントの回答をトランシーバーで自動読み上げする設定です。

- **ON**: メッセージ送信時にトランシーバーが自動接続され、AIの回答が音声で読み上げられます。30秒間新しい回答がなければ自動切断されます。
- **OFF**（デフォルト）: 読み上げは行われません。

!!! note
    この設定は端末のローカルストレージに保存されます。別の端末からログインした場合は再設定が必要です。

読み上げ機能の詳細は [TTS 読み上げ](../advanced/tts.md) を参照してください。
