# ユーザー管理

管理者は `/admin` ダッシュボードからユーザーを管理できます。

## 管理画面へのアクセス

管理者ロール（`admin`）を持つユーザーのみアクセス可能です。

## ユーザー一覧

登録済みのすべてのユーザーが一覧表示されます。

## ユーザーの作成

| 項目 | 説明 |
|---|---|
| ユーザーID | ログインに使用するID（一意） |
| 表示名 | チャットに表示される名前 |
| パスワード | 初期パスワード |
| ロール | `admin`（管理者）または `user`（一般） |

### API

```
POST /api/admin/users
```

```json
{
  "login_id": "user001",
  "display_name": "山田太郎",
  "password": "initial_password",
  "role": "user"
}
```

## ユーザーの編集

表示名やロールを変更できます。

### API

```
PUT /api/admin/users/:id
```

## ユーザーの有効化/無効化

ユーザーを削除する代わりに、アカウントを無効化できます。無効化されたユーザーはログインできなくなります。

### API

```
PATCH /api/admin/users/:id/status
```

```json
{
  "is_active": false
}
```

!!! note "Botユーザー"
    AIエージェント用のBotユーザーは `is_bot` フラグで識別されます。Botユーザーを作成する場合は、通常のユーザー登録の後にデータベースで `is_bot = true` を設定してください。
