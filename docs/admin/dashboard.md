# ダッシュボード

Tealus のシステムダッシュボード（`/system`）は、管理者がエージェントの設定やシステムの状態を管理するための画面です。

## アクセス

- URL: `/system` パスからアクセス
- 認証: Tealus ServerのJWTを共有（adminロール必須）

## 画面構成

| ページ | 説明 |
|---|---|
| 概要 | システムの稼働状況サマリー |
| ルーム管理 | ルーム一覧と状態確認 |
| エージェント設定 | AIエージェントの設定管理 |
| モニタリング | エージェントの稼働状態 |
| ログビューア | サーバーログの確認 |

## 技術仕様

ダッシュボードは `tealus/dashboard/` にReact + Viteで実装された独立したSPAです。PC向けの設計で、Tealus Server から `/system` パスで静的配信されます。
