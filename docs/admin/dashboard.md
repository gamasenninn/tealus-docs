# ダッシュボード

Tealus のシステムダッシュボード（`/system`）は、管理者がエージェントの設定やシステムの状態を管理するための画面です。

## アクセス

- URL: `/system` パスからアクセス
- 認証: Tealus ServerのJWTを共有（adminロール必須）

## 画面構成

| ページ | 説明 |
|---|---|
| 概要 | システムの稼働状況サマリー |
| ルーム管理 | ルーム一覧と状態確認 |
| エージェント設定 | AIエージェントの設定管理 |
| モニタリング | エージェントの稼働状態 |
| ログビューア | サーバーログの確認 |

## エージェント管理

### エージェント一覧

登録済みのエージェント（Botユーザー）が一覧表示されます。

| 表示項目 | 説明 |
|---|---|
| 表示名 | エージェントの表示名 |
| Login ID | ユーザーID |
| 状態 | 有効/無効バッジ |
| 作成日 | 登録日 |
| ルーム設定 | ルーム個別設定へのリンク |

右上の「グローバル設定」ボタンから、全エージェント共通の設定（`settings.json`・`mcp_config.json`）を編集できます。

### ルーム一覧

エージェントごとに、参加しているルームの一覧が表示されます。

| 表示項目 | 説明 |
|---|---|
| ルーム名 | トークルーム名 |
| タイプ | dm / group |
| メンバー数 | 参加メンバー数 |
| 応答モード | 現在の応答モード |
| 設定 | ルーム個別設定への歯車アイコン |

### ルーム個別設定

各ルームに対して4つのタブで設定を管理します。

#### 基本設定タブ

| 設定項目 | 説明 |
|---|---|
| エージェント有効/無効 | ルームでのエージェント応答の有効/無効 |
| 応答モード | 自動 / 全応答 / メンションのみ / OFF |
| 読み上げ音声モデル | TTS の音声モデル選択（10モデルから選択） |

#### Light プロンプトタブ

Light Agent に対する追加指示をテキストエリアで編集します。ルームごとにエージェントの性格や応答スタイルをカスタマイズできます。

#### Deep プロンプトタブ

Deep Agent（Claude Code）のシステムプロンプト（`CLAUDE.md`）を編集します。

#### MCP設定タブ

ルーム固有の MCP サーバー設定を JSON で直接入力します。

!!! note
    設定の変更は `room_settings.json` に保存されます。

## 技術仕様

ダッシュボードは `tealus/dashboard/` にReact + Viteで実装された独立したSPAです。PC向けの設計で、Tealus Server から `/system` パスで静的配信されます。
